# Notes App – Kotlin + Jetpack Compose + Supabase

A simple yet powerful **Notes** application built with **Android (Kotlin + Jetpack Compose)** and powered by **Supabase** as the backend.

This project demonstrates **CRUD** operations (Create, Read, Update, Delete) for personal notes using Supabase's PostgreSQL database and authentication.

https://github.com/basitbhatti/Supabase-Notes-App

## ✨ Features (current)

- User **authentication** (sign up & sign in with email)
- Create, read, update, and delete personal notes
- Clean Material 3 UI with Jetpack Compose
- MVVM architecture + StateFlow for reactive UI
- Supabase Kotlin SDK (`supabase-kt`) for auth & database

## 🚧 Work in Progress

I'm actively working on this project to master Supabase. Planned improvements include:

- Real-time note sync (using Supabase Realtime)
- Note categories / tags
- Search & filtering
- Offline support (with local caching)
- Image attachments via Supabase Storage
- Dark mode polish & animations

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Architecture**: MVVM + StateFlow
- **Backend**: Supabase (Auth + PostgREST)
- **Networking**: Supabase-Kt (io.github.jan-tennert.supabase)
